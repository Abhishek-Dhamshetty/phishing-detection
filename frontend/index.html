About This System

How It Works

  • URL Analysis: Examines URL structure, domain, protocol, special characters
  • HTML Analysis: Analyzes forms, input fields, links, scripts, keywords
  • Machine Learning: Uses a Random Forest classifier trained on phishing indicators
  • Risk Scoring: Provides confidence scores and identifies suspicious features

Key Detection Features

🔗 URL Features

  • Domain and URL length
  • Special characters (@, //)
  • IP addresses in URL
  • HTTPS presence
  • Subdomains

📄 HTML Features

  • Number of forms and input fields
  • Password fields
  • Form action URLs
  • Suspicious keywords
  • External links

⚠️ Risk Indicators

  • Mismatched domains
  • No HTTPS connection
  • Multiple password fields
  • Redirect characters
  • Unusual ports

Limitations

  • Detection accuracy depends on training data
  • Sophisticated phishing pages may evade detection
  • JavaScript-heavy pages require separate HTML analysis
  • Use in combination with browser security features

Technologies Used

  • scikit-learn - Machine Learning
  • BeautifulSoup - HTML parsing
  • Vanilla JavaScript - Frontend
